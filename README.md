# Shadow Circuit Fighters

A fully functional 2D fighting game built with Python and Pygame.

## Features
- 4 unique characters with complete backstories
- 2-player local multiplayer
- Special moves, combos, blocking, and hitstun
- Character selection with story mode
- Screen shake and particle effects

## Controls
- **Player 1:** WASD (move), G (attack), H (special), S (block)
- **Player 2:** Arrow keys (move), L (attack), K (special), DOWN (block)

## How to Run
1. Install Python 3.x
2. Install pygame: `pip install pygame`
3. Run: `python shadow_circuit_fighters.py`

## Characters
- **Kira Vance** (The Cyber-Sentinel) - Balanced fighter
- **Magnus Thorne** (The Iron Judge) - Heavy hitter
- **Zara Chen** (The Phantom Broker) - Fast and agile
- **Rocco "Bull" Marchetti** (The Reformed Enforcer) - Tanky bruiser
